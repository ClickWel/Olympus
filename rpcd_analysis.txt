# rpcd config analysis

/etc/config/rpcd contains:
```
option password '$p$root'
```

In OpenWrt ubus/rpcd, the $p$username syntax means "use the system password for username".
So $p$root = use root's system password.

Since root has no password set (shadow shows empty field), the effective password is nothing.
